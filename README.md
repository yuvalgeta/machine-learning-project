# Machine Learning Regression Project

A practical machine learning project focused on regression, built with Python and Jupyter Notebook. The project demonstrates an end-to-end workflow: understanding the problem, preparing the data, exploring the dataset, training a model, and evaluating results.

## Project Goal

The goal of this project is to predict house values using the California Housing dataset. It was created as a learning project to strengthen core machine learning skills and show a structured approach to solving a regression problem.

## What’s Included

- Data loading and initial inspection
- Data cleaning and preprocessing
- Exploratory data analysis
- Correlation analysis and visualization
- Regression modeling
- Model evaluation using standard metrics
- Interpretation of the results

## Dataset

This project uses the California Housing dataset, which contains numeric features such as:

- Median income
- House age
- Average rooms
- Average bedrooms
- Population
- Average occupancy
- Latitude
- Longitude

The target variable is the median house value.

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/yuvalgeta/machine-learning-project.git
   cd machine-learning-project
   ```

2. Open the notebook inside the `linear regression flow` folder.

3. Install the required packages if needed:
   ```bash
   pip install pandas numpy scikit-learn matplotlib seaborn scipy jupyter
   ```

4. Run the notebook cells from top to bottom.
